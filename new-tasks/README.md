# 치크 커스텀 훈련장 숏폼 프로젝트

배틀그라운드 커스텀 훈련장 콘텐츠 3편 제작을 위한 작업 공간입니다.

## 목표

- 3편 시리즈 숏폼 영상 생성 (각 18~19초)
- Veo 3.1-fast를 활용한 영상 생성
- 시리즈 일관성 유지 (동일 캐릭터, 세계관)

## 제약 사항

| 항목 | 처리 방식 |
|------|-----------|
| 자막/말풍선 | 후편집 (Veo 한글 깨짐 이슈) |
| 인터랙티브 요소 | 후편집 |
| 효과음 | 별도 추가 (삐약삐약 제외) |

**중요**: Veo 3.1-fast는 한글 텍스트가 깨지므로, 영상 내 텍스트는 생성하지 않습니다.

---

## 작업 흐름

```
1. 프롬프트 실험 (웹 UI)
   └── 캐릭터 참조 이미지 + 씬별 프롬프트 테스트
   └── 결과 확인 후 개선점 도출

2. 프롬프트 템플릿 확정
   └── 3편 모두에 적용 가능한 구조화된 템플릿

3. 코드 구현 (필요시)
   └── 기존 game_services.py 확장 또는 신규 타입 추가
```

## 세계관

### 등장인물

| 캐릭터 | 역할 | 참조 이미지 |
|--------|------|-------------|
| 게이머 | 주인공 | `assets/게이머.png` |
| ChatGPT | 로봇 조력자 | `assets/헬멧남.png` |
| 치크 | 병아리 조력자 | `assets/치크.png` |

### 배경

- 게이머의 집 (모니터 앞)
- 배틀그라운드 게임 내부
  - 도시 (건물, 거리)
  - 초원 (보급품 낙하 지역)
  - 창고 (실내 전투)

### EP 01: 자꾸 죽어서 챗 GPT한테 커스텀 훈련장 만들어달라 했다 (18초)

| 씬 | 시간 | 길이 | 배경 | 핵심 액션 |
|----|------|------|------|-----------|
| 1. 훅 | 00:00~00:05 | 5초 | 게이머의 집 | 모니터 속으로 빨려 들어감 |
| 2. 게임 입장 | 00:05~00:08 | 3초 | 도시 | 게이머 당황, GPT 설명 |

| 4. 위기 | 00:13~00:15 | 2초 | 도시 | 저격당함 |

**대사 요약**:
- 게이머: "으아 아니 진짜 게임속으로 들어가면 어떻게 하냐고!"

- 게이머: "맨날 벽만 쏘다가, 진짜 실전 같이 쏘니까 실력이 느는 것 같은데?"
- GPT: "그래도 방심은 금물입니다."
- 게이머: "으악! 빨리 숨어야해, 어디로 숨지?"
