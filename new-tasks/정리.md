# [숏폼] 커스텀 훈련장 관련 콘텐츠 기획안

## 1. 개요

- **제안배경**: 유튜브 영상 중 신규 시청자 수가 가장 많은 영상이 커스텀 훈련장 숏츠
- **참고사항**: 배그 자체 훈련장 시스템이 좋지 않고, 실력을 늘리고 싶은 초기 유저들도 커스텀 훈련장에 대한 니즈가 높음
- **적용사항**: 커스텀 훈련장을 챗 gpt에 의뢰 → 실제 게임속에서 실력 up → 치크를 만나 게임 신청
- **타깃**: 배그 실력을 꾸준히 하고 있는데 실력을 올리고 싶은 사람 / 배그를 나좀한다는 사람들

---

## 2. 가설 설정

| 항목        | 내용                                                     |
| ----------- | -------------------------------------------------------- |
| 항목        | 배그 커스텀 훈련장에 대한 니즈가 높을 것이다.            |
| 성공가설    | 커스텀 훈련장 관련 콘텐츠 3편 업로드시, 댓글수 증가할 것 |
| 실패가설    | 댓글수가 낮다면, 공감구조가 약했을 가능성이 높다.        |
| 후속 대응   | 다음 실험에서 '공감 메시지' 변경                         |

---

## 3. 세계관 세팅

- **등장인물**: 게이머(주인공), ChatGPT(로봇), 치크(조력자 병아리)
- **배경**: 게이머의 집, 배틀그라운드 게임 내부 도시 및 초원

---
